Simple web based tools that are ready to use in your browser.

Capture The Flag github.com/janosgyerik/capture-the-flag

Remaster Linux Live CD images for the purpose of creating ready to use security wargames with pre-installed vulnerabilities to exploit. https://sourceforge.net/projects/ctfomatic/files/

Source Bugs

jQuery Upvote janosgyerik.github.io/jquery-upvote

A jQuery plugin to generate a voting widget like the one used on Stack Exchange sites. It implements the same voting rules as on Stack Exchange, with callback option to trigger server methods via AJAX, and multiple widget styles.

Source Bugs

Google Map Tools janosgyerik.github.io/gmaptools

Find a location by its coordinates or address. Find the coordinates of any place on the map. More tools not available on Google Maps directly.

Source Bugs

Highlighter janosgyerik.github.io/highlighter

Highlight multiple keywords in any (copy & pasted) text, using different color for each search term. Remembers recently used keywords in Local Storage.
